# Gerald the Hermit Crab

Gerald the Hermit Crab is a simple 2D platformer that was created for a class project. It was developed on a short time frame, so it may have buggy, unoptimized, or messy code.

## Features

- **Basic player movement**: Left, right, jump
- **Dynamic camera**: Follows the player
- **Physics**: Collisions with environment and enemies
- **Mario-like combat with enemies**: Jump on their head to kill
